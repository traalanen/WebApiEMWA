EMWA

Every Minute Workout Assistant

EMWA is a simple cardio and HIIT (High Intensity Interval Training) Timer and Workout Log.
This application is a SPA (Single Page Application) and a work in progress. It does not currently save the user's workout log beyond the current session.
A future iteration will use Azure or AWS services for maintaining data.
